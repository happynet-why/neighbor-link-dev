CityLink - Setup Access to Starlink via Iran Internet
Dashboard
Configuring Remote Access (CityLink): You can now set up your system to access your autonomous internet source (e.g., Starlink) remotely through the Iranian internet. We refer to this feature as "CityLink." Please carefully follow the steps provided for a successful setup. If you do not require remote access to your Starlink connection, feel free to skip this step
Instruction:
  • Order an Iran-based VPS running Ubuntu 22.04 or a later version.
  • SSH into the server and execute the following command for server preparation:
    sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/nasnet-community/neighbor-link/refs/heads/main/server/Iran_server/Iran_server_setup.sh)"
  • Copy the installation output and paste it into the connection string section.
  • Click "Connected" and wait for the connection to be established.
  • Once the connection is established, you will see options to choose between Proxy or Outline solution.
